# 데이터 시각화 강의 노트

## 1. 시각화의 기본 개념 (Data Visualization Basics)

### 개요

* **데이터의 포인트:** 시각화 기본 개념, Matplotlib, Seaborn, Plotly 활용

### 탐색적 데이터 분석(EDA)에 필요한 시각화

* **목적:** 데이터의 분포 확인, 특징값 발견, 변수 간 관계 분석 등을 위해 알맞은 시각화 방식을 선택해야 함.

### 데이터 종류에 따른 그래프 시각화 방법

| 시각화 유형 | 설명 | 용도 |
| --- | --- | --- |
| **히스토그램 (Histogram)** | 데이터의 도수 분포를 나타냄. (X축: 구간, Y축: 데이터 개수) | 데이터의 분포 확인, 특정 값이 많이 나타나는지 분석 |
| **막대 그래프 (Bar Chart)** | 항목별 차이를 나타낼 때 사용. (세로 막대) | 시간 흐름이나 순차적인 데이터 표현, 항목별 비교 |
| **산포도 (Scatter Plot)** | 복수의 변수를 비교하거나 상관관계를 파악할 때 사용 | 변수 간 관계 분석, 상관관계 확인, 데이터 분포 패턴 분석 |
| **박스 플롯 (Box Plot)** | 데이터의 분포와 이상치를 시각화 (중앙값, 사분위수 등) | 데이터 분포 비교, 이상치 탐지 |
| **바이올린 플롯 (Violin Plot)** | 박스 플롯 + KDE(커널 밀도 추정) 결합 | 데이터의 밀도와 분포 형태 분석 |
| **히트맵 (Heatmap)** | 행렬 형태의 데이터를 색상으로 표현 | 상관관계 분석, 행렬 데이터 시각화 |
| **버블 차트 (Bubble Chart)** | 산포도와 유사하나 점의 크기를 변수로 활용 | 3차원 데이터 시각화, 변수 간 관계 분석 |

## 2. Matplotlib을 활용한 시각화

### Matplotlib의 장단점

* **장점:** 유연성(모든 종류의 plot 가능), 넓은 사용자층과 문서, 정교한 제어 가능
* **단점:** 복잡성(간단한 작업도 많은 코드 필요), 기본 스타일이 투박함(추가 스타일링 필요)

### pyplot의 중요 요소

* **Figure:** 그림을 그리기 위한 Canvas 역할, 그림판의 크기 등을 조절
* **Axes:** 실제 그림을 그리는 메소드들을 가짐 (x축, y축, Title 등의 속성 설정)

### 코드 예제 및 활용

#### 1) 기본 plot 그리기

```python
import matplotlib.pyplot as plt

# 리스트가 하나면 y값으로 인식, x축은 자동(0,1,2...)
plt.plot([1, 2, 3, 4]) 
plt.ylabel('some numbers') # y축 라벨 설정
plt.show()

```

#### 2) 그래프 formatting 변경

```python
import matplotlib.pyplot as plt

# 'ro'는 빨간색 원형 마커를 의미
plt.plot([1, 2, 3, 4], [1, 4, 9, 16], 'ro') 
# 축 범위 설정 [xmin, xmax, ymin, ymax]
plt.axis([0, 6, 0, 20]) 
plt.show()

```

#### 3) 여러 개의 plot을 가지는 Figure

```python
import matplotlib.pyplot as plt

# nrows=1, ncols=2: 1행 2열의 서브플롯 생성
fig, (ax1, ax2) = plt.subplots(nrows=1, ncols=2, figsize=(10, 6))

```

#### 4) y축의 scale 변경

```python
import numpy as np
import matplotlib.pyplot as plt

# 데이터 생성
np.random.seed(19680801)
y = np.random.normal(loc=0.5, scale=0.4, size=1000)
y = y[(y > 0) & (y < 1)]
y.sort()
x = np.arange(len(y))

# 다양한 스케일 적용 예시
plt.yscale('linear')  # 선형
plt.yscale('log')     # 로그
plt.yscale('symlog', linthresh=0.01) # 대칭 로그
plt.yscale('logit')   # 로짓
plt.show()

```

#### 5) 여러 구성 요소 활용 (라벨, 타이틀, 범례)

```python
import numpy as np
import matplotlib.pyplot as plt

x_value = np.arange(0, 100)
y_value = 2 * x_value
y_value2 = 4 * x_value # 화면상 4배 예시 추정

plt.plot(x_value, y_value, color='green', label='LINE1') # 라벨 설정
plt.plot(x_value, y_value2, color='blue', label='Line2')

plt.xlabel('X Axis') # X축 이름
plt.ylabel('Y Axis') # Y축 이름

# X축 눈금을 5 간격으로 설정 & 90도 회전
plt.xticks(ticks=np.arange(0, 100, 5), rotation=90) 

plt.title('Hello Ssafy') # 타이틀
plt.legend() # 범례 추가
plt.show()

```

## 3. Seaborn을 활용한 시각화

### Seaborn의 특징

* **Figure level 함수:** Figure 전체를 관리 (여러 그래프 관리 용이) - `relplot`, `displot`, `catplot`
* **Axes level 함수:** 개별 Axes에서 하나의 플롯을 그리는 데 사용 - `scatterplot`, `histplot`, `stripplot` 등

### 코드 예제

#### 1) 그래프 크기 및 색상 설정

```python
# 그래프 크기 설정
plt.figure(figsize=(8, 5))

# 색상 설정 예시
colors = ["#FF0000", "#00FF00", "0000FF"] # 예시 색상 코드

```

#### 2) Seaborn Scatterplot 활용

```python
# 흡연 여부(smoker)로 색상 구분, 식사 시간(time)으로 점 스타일 구분
sns.scatterplot(data=df, x="total_bill", y="tip", hue="smoker", style="time", 
                deep=5, markers={"Lunch": "o", "Dinner": "X"})

plt.title("Total Bill vs Tip (색상: 흡연 여부, 스타일: 식사 시간)")
plt.xlabel("총 지불 금액 ($)", fontsize=14)
plt.ylabel("팁 ($)", fontsize=14)
# 범례 위치 설정
plt.legend(loc="upper left") 

```

## 4. Plotly를 활용한 시각화

### Plotly의 장단점

* **장점:** 상호작용성(Interactive), 웹 통합 최적화, 3D 및 지리적 시각화 강력, 세련된 기본 그래프 제공
* **단점:** 학습 곡선 존재, 대규모 데이터 처리 시 성능 문제 발생 가능, 세밀한 조정이 복잡할 수 있음

### 코드 예제

#### 1) 기본 Plot 활용 및 템플릿

```python
import plotly.express as px

# 갭마인더 데이터 로드
df = px.data.gapminder()

# 2007년 데이터 필터링
df_2007 = df.query("year==2007")

# 다양한 Plotly 템플릿을 적용하여 시각화
# 템플릿 예: "plotly", "plotly_white", "plotly_dark", "ggplot2", "seaborn", "simple_white", "none"

for template in ["plotly", "plotly_dark"]:
    fig = px.scatter(df_2007, 
                     x="gdpPercap", y="lifeExp", size="pop", color="continent",
                     log_x=True, size_max=60,
                     template=template, title="Gapminder 2007")
    fig.show()

```

#### 2) 애니메이션 및 차원 추가 (화면 캡처 기반 추론)

* `animation_frame="year"` 등을 활용하여 시계열 변화를 애니메이션으로 표현 가능 (화면의 버블 차트 예시).
