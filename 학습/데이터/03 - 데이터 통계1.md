## 2. 기초 통계량 (Basic Statistics)

데이터를 대표하는 값을 구하여 데이터의 전반적인 특성을 살펴보는 방법입니다.

### 2.1 위치 (Location) - 중심 경향성

* **평균 (Mean):** 모든 값을 더해 개수로 나눈 값. 전체적인 중심을 나타내지만 이상치(Outlier)에 민감함.
* **절사 평균 (Trimmed mean):** 상·하위 극단값 일부를 제거해 계산한 평균. 이상치 영향을 줄일 때 유용.
* **가중 평균 (Weighted mean):** 각 값의 중요도(가중치)를 반영해 계산하는 평균. 값의 영향력이 동일하지 않을 때 사용.
* **중간값 (Median):** 정렬된 데이터의 중앙 위치 값. 왜곡된 분포에서도 안정적으로 중심을 나타냄.
* **중가중값 (Weighted median):** 값의 가중치를 고려해 중앙 위치를 결정하는 지표. 표본 비중이 다른 조사에서 활용.
* **백분위수 (Percentile):** 데이터를 100등분했을 때 특정 위치의 값으로 분포의 형태와 위치를 파악하는 데 사용.

### 2.2 변이 (Variability) - 퍼짐 정도

* 데이터가 얼마나 퍼져 있는지를 확인하는 지표입니다.
* **분산 (Variance):** 평균으로부터 얼마나 퍼져 있는지를 나타내는 지표.
* **표준편차 (Standard deviation):** 데이터의 편차(각 값에서 평균을 뺀 값)의 기준이 되는 값.
* **사분위범위 (IQR):** 25번째 백분위수와 75번째 백분위수의 차이를 보는 것.

### 2.3 기초 통계량의 활용

* **이상치 (Outlier) 발견:**
* 데이터를 분석할 때 극단적으로 크거나 작은 값.
* 이상치는 평균-분산을 크게 왜곡시킴.
* *예: 평균 몸무게가 60kg인데 120kg이 포함되면 이상치 가능성.*


* **비교와 예측 가능:**
* 두 개 이상의 그룹을 비교할 때 평균과 표준편차를 이용.
* *예: 남학생과 여학생의 키 분포 비교.*


* **적절한 분석 방법 선택:** 데이터의 분포 형태에 따라 분석 방법 결정.

## 3. 조건부 확률 (Conditional Probability)

### 3.1 개념 및 공식

* **개념:** 어떤 사건(정보 B)이 주어졌을 때, 그 조건 아래에서 다른 사건(A)이 일어날 확률.
* **공식:**


* 사건 B가 이미 일어났다는 조건을 알고 있을 때, 그 정보를 반영해서 A가 일어날 가능성을 나타내는 확률.

### 3.2 베이즈 정리 (Bayes' Theorem)

* 직접 구하기 어려운 $P(A|B)$를, 더 구하기 쉬운 , , $P(B)$로 계산할 수 있게 해주는 공식.
* 조건부 확률의 방향을 뒤집어 주는 역할.
* 데이터 분석 및 머신러닝에서 자주 쓰이는 개념.
* **공식:**


* 현실에서 측정 가능한 확률()을 통해 우리가 알고 싶은 확률()을 추론.

### 3.3 나이브 베이즈 분류 (코드 예시)

* 추후 나이브 베이즈라는 방식에서 적용될 수 있음.

```python
# 관찰 가능한 확률들
P_word_given_spam = 0.7  # 스팸 메일 중 특정 단어가 등장할 확률
P_word_given_normal = 0.1 # 정상 메일 중 단어 등장 확률
P_spam = 0.4             # 전체 메일 중 스팸 비율
P_normal = 0.6           # 전체 메일 중 정상 비율

# 전체 단어 등장 확률
P_word = P_word_given_spam * P_spam + P_word_given_normal * P_normal

# 우리가 알고 싶은 확률: 단어가 등장했을 때 스팸일 확률
print(f"P(spam | word): {P_spam_given_word}")

```

## 4. 표본 추출과 표본 편향 (Sampling and Bias)

### 4.1 표본 조사의 필요성

* **인구 조사 (Census) vs 표본 조사**
* **인구 조사:** 전체 국민을 조사하는 방법 (모집단 조사). 가장 정확하지만 시간과 비용이 많이 소요됨.
* **여론 조사 (Survey)  표본 조사:** 모든 사람을 조사하는 것이 어렵기 때문에 일부만 조사하여 전체를 추론.
* *예시: 21대 총선 정당후보 지지도 조사 (성인 남녀 1,011명 응답).*



### 4.2 추출 (Sampling)의 개념

* 모집단에서 일부를 **선택(추출)**하는 과정.
* **용어:**
* **모집단:** 전체 대상 (예: 떡국 전체).
* **표본:** 추출된 일부 (예: 국물을 맛보기 위해 뜬 한 숟가락).



### 4.3 표본 추출 방법

| 방법 | 설명 | 예시 |
| --- | --- | --- |
| **단순 무작위 추출** | 모든 사람이 동일한 확률로 뽑힘 | 모자 속에서 랜덤하게 종이 한 장 뽑기 |
| **층화 추출** | 모집단을 그룹(층)으로 나눈 후, 각 층에서 무작위로 표본 추출 | 학년별 반 대표 한 명씩 뽑기 |
| **계통 추출** | 모집단을 일정 간격으로 나누어 표본을 추출 | 첫 번째 학생을 랜덤 선발 후, 5명마다 한 명씩 조사 |
| **군집 추출** | 모집단을 여러 그룹(군)으로 나눈 후, 몇 개의 군을 랜덤 선택 | 한 반을 통째로 뽑아 조사 |
| **편의 추출** | 쉽게 접근할 수 있는 표본을 선택 (비추천) | 길거리에서 가까운 사람만 설문조사 |

### 4.4 임의 추출(무작위 추출)의 개념

* 모집단에서 특정한 규칙 없이 **무작위로 표본을 선택**하는 방법.
* **공정한 위치:** 모든 대상이 동일한 확률로 선택될 기회 제공.
* **편향 방지:** 특정 그룹에 치우치지 않고 모집단 전체를 반영.
* **통계 분석의 신뢰도 증가:** 모집단을 잘 반영한 표본이면 정확한 결과를 얻을 가능성이 높음.

### 4.5 표본 편향 (Sampling Bias)

* **개념:** 모집단을 대표하지 못하는 **잘못된 표본**을 선택하여 조사 결과가 왜곡된 현상.
* **핵심:** 대표성을 담보하려면 **임의 표본 추출**이 중요.
* **사례: 루즈벨트의 당선 예측 (리터러리 다이제스트)**
* 240만 명 대상 여론조사 결과 랜드(Landon) 승리 예측  실제로는 루즈벨트 압승.
* **원인:** 부유한 사람들(자동차, 전화기 보유자) 위주로 표본이 포함되어 서민층을 배제함(잘못된 여론 조사).



### 4.6 표본 편향을 줄이는 방법

* **가중치 (Weighting) 적용:**
* 특정 그룹이 과소/과대 대표된 경우 보정하는 방법.
* 부족한 그룹의 응답을 보정하기 위해 가중치를 적용해 결과를 반영.



### 4.7 편향의 차이점 (표본 편향 vs 선택 편향)

| 구분 | 설명 | 특징 |
| --- | --- | --- |
| **표본 편향** (Sampling Bias) | 표본을 **선택하는 방식** 때문에 생기는 편향 | 연구자가 표본을 잘못 설계하여 과대/과소 대표됨. |
| **선택 편향** (Selection Bias) | 표본이 **스스로 선택**하는 방식 때문에 생기는 편향 | 자발적 참여 등으로 특정 성향의 사람만 포함됨. (예: 건강에 관심 있는 사람만 건강검진 참여) |

### 4.8 표본 편향 예제 (타이타닉 데이터셋 코드)

```python
# 전체 승객의 등급별 생존률 계산
survival_rates_by_class = titanic.groupby('class', observed=False)['survived'].mean()

# 각 등급별 생존률 계산
print(survival_rates_by_class) 
# (출력 결과 예시: First 0.629, Second 0.472, Third 0.242)

# 시각화: 등급별 생존률 막대 그래프
plt.figure(figsize=(8, 5))
sns.barplot(
    x=survival_rates_by_class.index,
    y=survival_rates_by_class.values,
    palette='coolwarm'
)
plt.title("Survival Rate by Class")
plt.show()

```

* **해석:**
* 전체 생존율은 약 38%.
* 1등석 생존율은 63%로 전체보다 매우 높음  1등석 승객만 표본으로 선택하면 생존율이 과대평가될 가능성 있음.
* 3등석 생존율은 24%로 전체보다 낮음  3등석만 선택하면 과소평가될 수 있음.

## 5. 분포에 대한 이해 (Understanding Distribution)

### 5.1 분포(Distribution)의 필요성

* **데이터의 패턴 파악:** 데이터가 대체로 어디에 몰려 있는지(중심경향) 확인.
* **이상치 발견:** 데이터를 분석할 때 극단적으로 크거나 작은 값(이상치)을 찾는 것이 중요.
* **비교와 예측 가능:** 두 개 이상의 그룹을 비교할 때 어떤 차이가 있는지 시각적으로 확인 가능.
* **적절한 분석 방법 선택:** 예) 정규분포(대칭형)라면 평균값을, 왜곡된 분포라면 중앙값이 더 적절함.

### 5.2 시각화 방법

* **히스토그램 (Histogram):** 데이터를 막대 그래프로 나타낸 것. 데이터의 전체적인 분포를 쉽게 확인 가능.
* **밀도 곡선 (Density Plot):**
* 데이터를 연속적인 곡선 형태로 나타내어 분포를 부드럽게 표현한 그래프.
* 히스토그램처럼 데이터를 나누지 않고 연속적인 곡선으로 표현.
