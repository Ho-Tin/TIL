# 데이터 분석을 위한 기초 통계 2

## 1. 분포에 대한 이해

### 확률분포 (Probability Distribution)

* **정의:** 데이터가 나타날 수 있는 모든 값과 그 값들의 나올 확률을 정리한 것.
* 예) 사람의 키, 시험 점수, 동전 던지기 결과 등


* **가치:**
* 데이터가 어느 값 주변에 얼마나 모여 있는가?
* 특정 값이 나타날 가능성은 어느 정도인가?


* **종류:**
* **연속형 분포:** 정규분포, 균등분포 등
* **이산형 분포:** 이항분포, 포아송분포 등


* **활용:** 데이터의 패턴과 특징을 설명하고, 예측 모델과 통계적 추론에서 적합한 분석 방법을 선택할 수 있음.

### 통계학에서의 표본 분포

* **표본 분포:** 모집단에서 표본을 여러 번 뽑아서, 그때마다 계산한 "평균, 비율" 같은 통계량이 만드는 분포.
* **특징:**
* 단일 표본 하나만 봤을 때는 샘플 평균이 모집단의 평균에 가까울 수도 아닐 수도 있음.
* 많은 표본 평균을 모으면 **특정한 패턴**을 띠게 됨.


* **예시:** 전국 고등학생의 평균 키 조사 (전수 조사가 어려우므로 표본 조사를 여러 번 반복하여 분포를 확인).

### 대표적인 분포의 종류

| 분포 유형 | 설명 | 활용 |
| --- | --- | --- |
| **정규분포** | 평균을 중심으로 대칭적인 종 모양 분포 | 키, 몸무게, 시험 점수 |
| **균등분포** | 모든 값이 같은 확률을 가짐 | 난수 생성, 시뮬레이션 |
| **이항분포** | 성공/실패 같은 이진 결과 모델링 | 이메일 클릭률, 제품 불량률 예측 |
| **포아송 분포** | 일정 시간/공간 내에서 사건 발생 횟수 모델링 | 콜센터 문의 수 |

## 2. 정규분포 (Normal Distribution)

* **정의:** 데이터가 평균을 중심으로 **대칭적인 종 모양**을 띠는 분포.
* **특징:**
* 평균()과 표준편차()로 정의됨.
* 자연 현상, 사회 현상 등 많은 데이터가 정규분포를 따름.


* **활용 (머신러닝):**
* 많은 알고리즘(예: 회귀 분석)이 데이터가 정규분포를 따른다는 가정하에 설계됨.
* 데이터의 복잡성과 불확실성을 효과적으로 관리 가능.



### [코드] 정규분포 확인 및 시각화

```python
import numpy as np
import matplotlib.pyplot as plt
import seaborn as sns

# 정규분포 데이터 생성 (평균=0, 표준편차=1, 샘플개수=1000)
# loc: 평균, scale: 표준편차, size: 샘플 수
data_normal = np.random.normal(loc=0, scale=1, size=1000)

# 시각화
plt.figure(figsize=(12, 6))
sns.distplot(data_normal, kde=True, color='lightgreen')
plt.title("Normal Distribution Visualization")
plt.show()

```

### QQ-plot (Quantile-Quantile Plot)

* **목적:** 데이터의 실제 분포가 정규분포와 얼마나 가까운지 그래프상으로 확인하는 방법.
* **해석:** 점들이 붉은색 대각선(직선) 위에 가까이 있으면 정규 분포에 가까운 것.

## 3. 중심극한정리 (Central Limit Theorem, CLT)

* **정의:** 모집단의 분포가 어떤 모양이든 상관없이, 표본의 크기()가 충분히 크다면 표본 평균들의 분포는 **정규분포에 가까워진다**는 원리.
* **의의:** 다양한 환경에서도 정규 분포를 활용한 분석이 가능하게 함.

### [코드] 중심극한정리 시뮬레이션

```python
import numpy as np
import matplotlib.pyplot as plt
import seaborn as sns

# 모집단이 균등분포라고 가정 (0~1 사이)
sample_size = 1000  # 표본의 크기
sample_means = []

# 여러 번 표본을 추출하여 평균을 계산
for _ in range(1000):
    sample = np.random.exponential(scale=1, size=sample_size) # 지수분포 예시
    sample_means.append(np.mean(sample))

plt.figure(figsize=(12, 6))
sns.distplot(sample_means, kde=True)
plt.title("Distribution of Sample Means (CLT)")
plt.show()

```

## 4. 기타 주요 분포

### 이항분포 (Binomial Distribution)

* 성공/실패 같은 **이진(두 가지) 결과**를 다루는 확률 모델.
* 독립적인 실험이 번 반복될 때 성공 횟수를 세는 것.
* **예시:** 동전 10번 던지기, 특정 숫자가 나오는 횟수.

### 포아송 분포 (Poisson Distribution)

* **일정한 시간 또는 공간**에서 사건이 발생하는 횟수를 모델링.
* **조건:** 사건 발생이 독립적이어야 하며, 발생 확률이 일정해야 함.
* **관계:** 시행 횟수가 많고 사고 발생 확률이 매우 작으면 이항분포는 포아송 분포에 근사함.
* **예시:** 은행 방문자 수, 하루 동안 발생하는 전화 문의 수.

---

## 5. t-분포 (Student's t-distribution)

* **배경:** 모집단의 표준편차를 모르거나 표본의 크기가 작을 때 사용.
* **특징:**
* 정규분포와 유사하지만 **꼬리 부분(Tail)이 더 두꺼움**.
* 이는 데이터가 평균에서 멀리 떨어진 값(극단값)이 나올 가능성이 더 높음을 의미(불확실성 반영).


* **자유도 (Degrees of Freedom):**
* 자유도가 커질수록(표본이 많아질수록) 정규분포와 비슷해짐.
* 자유도가 작을수록 꼬리가 두꺼워짐.

## 6. 가설 검정과 신뢰 구간

### 통계적 실험의 중요성

* 데이터를 분석하지 않고 직관이나 경험에 의존하면 잘못된 결론을 내릴 위험이 큼.
* **예시:** 신약 개발 시 효과가 우연인지 실제인지 검증 필요.

### 유의성 검정 (Significance Testing)

* "우리가 관측한 차이가 우연히 발생했을 확률"을 계산.
* **p-value:** 우연히 이런 결과가 나올 확률.
* p-value < 0.05 : 우연이 아닐 확률이 높음 (통계적으로 유의미함).



### 신뢰 구간 (Confidence Interval, CI)

* 모집단의 참된 평균(모수)을 포함할 것으로 예상되는 값의 범위.
* **95% 신뢰구간:** "새로운 마케팅 전략이 구매율을 평균 2~5% 증가시킬 것이다" (95% 확신).
* 0을 포함하지 않으면 효과가 유의미하다고 판단하는 경우가 많음.

### 가설 검정의 개념

* **귀무가설():** "차이가 없다" 또는 "효과가 없다"는 기본 가설.
* **대립가설():** "차이가 있다" 또는 "효과가 있다"는 입증하려는 가설.
* **유의수준():** 1종 오류(참인데 기각하는 오류)를 허용할 최대 확률 (보통 0.05).

### t-통계량

* 두 집단의 평균 차이가 우연인지, 실제 의미 있는 차이인지를 판단하기 위한 지표.
* 공식: 
* 평균 차이가 클수록(분자), 데이터가 덜 퍼져있을수록(분모) t값은 커짐.

### p-value 해석

* : 귀무가설 기각  "차이가 유의미하다!"
* : 귀무가설 유지  "차이가 우연일 가능성이 크다."

### 가설 검정 프로세스

1. 가설 설정 ()
2. 유의수준 결정 (보통 )
3. 검정 통계량 계산 (t-검정 등)
4. p-value 확인
5. 결론 도출 (가설 채택 여부 결정)

## 7. 가설 검정 실습 (Python Code)

### [코드] 단측 검정과 양측 검정 실습

```python
import numpy as np
import scipy.stats as stats
import matplotlib.pyplot as plt
import seaborn as sns

# 데이터 생성 (A지구 vs B지구 승용차 이용률 비교 예시)
np.random.seed(1)
a_district = np.random.normal(loc=70, scale=10, size=30)
b_district = np.random.normal(loc=60, scale=10, size=30) # 평균 차이 존재

# 시각화
plt.figure(figsize=(10, 5))
sns.kdeplot(a_district, label='District A', shade=True)
sns.kdeplot(b_district, label='District B', shade=True)
plt.legend()
plt.show()

# 정규성 검정 (Shapiro-Wilk test)
# p-value > 0.05 이면 정규성을 만족한다고 가정
print(stats.shapiro(a_district))
print(stats.shapiro(b_district))

# 등분산 검정 (Levene's test)
# 두 집단의 분산이 같은지 확인
levene_test = stats.levene(a_district, b_district)
print(f"Levene Result: {levene_test}")

# 독립표본 t-검정 (Two-sample t-test)
# equal_var=True (등분산 가정 시)
t_stat, p_val = stats.ttest_ind(a_district, b_district, equal_var=True)

print(f"t-statistic: {t_stat:.4f}")
print(f"p-value: {p_val:.4f}")

# 결론 도출
alpha = 0.05
if p_val < alpha:
    print("귀무가설 기각: 두 집단의 평균 차이는 유의미함.")
else:
    print("귀무가설 채택: 두 집단의 평균 차이는 우연일 가능성이 높음.")

```

**실습 결과 예시 (화면 내용):**

* 독립표본 t-검정 결과
* t-value: -4.5129
* p-value: 0.0000 (0.05보다 작으므로 차이가 유의미함)
